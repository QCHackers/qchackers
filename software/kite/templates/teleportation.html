Quantum Teleportation Simulation

Quantum teleportation is a process by which quantum information from one location to another, with the help of classical communication and previously shared quantum entanglement between the sending and receiving location. When you click "Teleport!" This machine runs Alice's code which generates the entangled state and measures the qubits stored in register 0 and 1. The measured values are now classical bits and sent to a separate server where Bob's code is run in our Quantum Virtual Machine. From the classical bits that are measured on Alice's machine and sent to Bob's we are able to impart the quantum state into qubit
